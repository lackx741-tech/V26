# FAQ

**Q:** Which wallets and networks are supported?  
**A:** All EVM-compatible chains and WalletConnect-friendly wallets.

**Q:** How do I add new admins?  
**A:** Use the Admin Panel > Users section to invite new admins.

**Q:** How do I troubleshoot connection issues?  
**A:** See [Troubleshooting](troubleshooting.md).
