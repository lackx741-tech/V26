# Platform Modules

## 1. GENERAL Module
Configures UI, branding, and interface behavior

## 2. WALLET Module
Sets wallet validation, chain interaction, access rules, and modal style

## 3. TOKENS Module
Handles ERC-20, NFT transaction flows, approval logic, batch execution

## 4. MODAL Module
Controls connection UI, modal appearance & WalletConnect integration

## 5. ADVANCED Module
Fine-tunes logging, analytics, signature policy, caching, and automation

See each module’s section in [API & SDK Reference](api_sdk_reference.md).
