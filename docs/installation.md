# Installation

## Prerequisites
- Domain assigned
- Hosting/server access
- Node.js and Database (MySQL/Postgres)

## Steps
1. Clone/download the code
2. Set up environment variables in `.env`
3. Install dependencies: `npm install`
4. Build and start: `npm run build && npm start`
