# Smart Contract Integration with StealthV26

## 1. Deploy Your Contracts

- Use Hardhat/Foundry/Remix.
- Save contract addresses, ABI files, and chain info.

## 2. Register Contracts in StealthV26

- Add contracts in Admin, `.env`, or config JS.
- Specify network, type, and ABI (for custom logic).

## 3. SDK/Frontend Integration

- Use SDK to call approve, transfer, mint, claim, multicall, etc.

## 4. Map Actions to UI

- Show supported actions in modal/buttons.

## 5. Security & Compliance

- Explicit methods/ABIs, user previews, EIP-712 only.
