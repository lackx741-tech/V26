# Troubleshooting

## Common Issues

- **Admin panel won't load:**  
  Check domain, environment variables, and server logs.

- **Database errors:**  
  Verify DB connection params and that the DB is accessible from your host.

- **Wallet not connecting:**  
  Ensure correct RPC endpoints, network IDs, and WalletConnect configuration.

- **Notifications not arriving:**  
  Double-check Telegram bot token, group/channel IDs, and test with the in-app feature.
